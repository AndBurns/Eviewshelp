@cos
Element Functions
Cosine of argument specified in radians.
Compute the cosine of x (specified in radians).
Syntax: @acos(x)
x: number
Return: number
Examples
= @cos(@pi)
returns -1.

See also @acos, @asin, @atan, @sin, and @tan.
