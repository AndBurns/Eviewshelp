@cvars
Matrix Column Statistics
Variance (sample) of each column of a matrix.
Syntax: @cvars(m)
m: matrix
Return: vector
Returns the column sample variance.
Examples
= @cvars(mat)
returns a (column) vector whose i-th element is the sample variance for the i-th column of MAT.
