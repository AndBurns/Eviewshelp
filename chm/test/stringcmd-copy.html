copy
String Procs
Creates a copy of the string.
Creates either a named or unnamed copy of the string.
Syntax
string_name.copy
string_name.copy dest_name
Examples
s1.copy
creates an unnamed copy of the string S1.
s1.copy s2
creates S2, a copy of the string S1.
