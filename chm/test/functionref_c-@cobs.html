@cobs
Matrix Column Statistics
Number of non-NA values in each column of a matrix.
Syntax: @cobs(m)
m: matrix
Return: vector
Returns a vector containing the number of non-missing values in each column of the matrix m.
Examples
Let M1 be an lower triangular matrix whose elements above the main diagonal are NAs. In this case,
= @cobs(m1)
returns a column vector whose elements are the integers , , ..., 1.
