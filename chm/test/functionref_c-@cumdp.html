@cumdp
Cumulative Statistics
Cumulative process of positive (above threshold) changes.
Compute the partial sum process of positive (above the threshold y) changes in the series x beginning in the specified date.
Syntax: @cumdp(x, d[, y, s])
x: series
d: string
y: (optional) number
s: (optional) sample string or object
Return: series
