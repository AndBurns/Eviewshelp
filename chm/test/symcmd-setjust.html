setjust
Sym Procs
Set the horizontal justification for all cells in the spreadsheet view of the sym object.
Syntax
sym_name.setjust format_arg
where format_arg may be set to left, center, right, or auto (strings are left-justified and numbers are right-justified). Default display settings can be set in General Options; see “Spreadsheet Data Display”.
Examples
sym1.setjust left
left-justifies the cells in the spreadsheet view of the sym SYM1.
Cross-references
See Sym::setwidth and Sym::setindent for details on setting spreadsheet widths and indentation.
