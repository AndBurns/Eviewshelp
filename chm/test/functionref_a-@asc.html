@asc
Element Functions | String Functions
String to ASCII value.
Syntax: @asc(str)
str: string
Return: string
Returns the integer ASCII value for the first character in the string str.
Examples
scalar s1 = @asc("C")
scalar s2 = @asc("c")
returns the scalars S1=67 and S2=99.
