@rtdist
Element Functions
Student’s distribution random draw.
Syntax: @rtdist(v)
v: number,
Return: number
Draw a random value from the Student’s distribution with probability density function,
Examples
= @rtdist(1)
returns a random draw from the Cauchy distribution.
