@dweib
Element Functions
Weibull probability density.
Syntax: @dweib(x, m, a)
x: number,
m: number,
a: number,
Return: number
Evaluates the probability density function,
for and 0 elsewhere.
Examples
= @dweib(@log(2), 1, 1)
returns 0.5.
Cross-references
See also @cweib, @qweib, and @rweib.
