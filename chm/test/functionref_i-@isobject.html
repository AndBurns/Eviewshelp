@isobject
Support Functions
Syntax: @isobject(str)
str: string
Return: integer
Check for an object’s existence in the current workfile page. Returns a “1” if the object with the name str exists in the current workfile, and a “0” if it does not exist.
Examples
scalar exists = @isobject("x")
