@log1p
Element Functions
Natural logarithm function of 1 plus argument.
Syntax: @log1p(x)
x: number
Return: number
Returns .
Provides higher precision calculation of the expression for near 0 than direct evaluation using
@log(1 + x)
Examples
= @log1p(0.05)
returns 0.04879....
Cross-references
See @log,, @log1p, @log10, @logx, @log2pi.
See also exp and @exp.
