Working with Tables and Spreadsheets
There are three types of tables in EViews: tabular views, which are tables used in the display of views of other objects, named table objects, and unnamed table objects. The main portion of this discussion focuses on the use of commands to customize the appearance of named table objects. The latter portion of the chapter describes the set of tools that may be used to customize the display characteristics of spreadsheet views of objects (see “Customizing Spreadsheet Views”).
You may use EViews commands to generate custom tables of formatted output from your programs. A table object is an object made up of rows and columns of cells, each of which can contain either a number or a string, as well as information used to control formatting for display or printing.
“Table and Text Objects” describes various interactive tools for customizing table views and objects.
