@cmin
Matrix Column Statistics
Minimal value for each column of the matrix.
Syntax: @cmin(m)
m: matrix
Return: vector
Returns a vector containing the minimal values of each column of the matrix m.
Examples
Let ID be an identity matrix. Then
= @cmin(id)
returns a column vector of zeros.
Cross-references
See also @cimax, @cimin, and @cmax.
