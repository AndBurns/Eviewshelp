@expm1
Element Functions
Exponential function minus 1.
Syntax: @expm1(x)
x: number
Return: number
Returns .
Provides higher precision calculation than direct evaluation using
@exp(x) - 1
for near 0.
Examples
= @expm1(0.05)
returns 0.05127....
