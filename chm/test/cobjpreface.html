Object Reference
The three sections of the EViews Object Reference consist of reference material for working with views and procedures of objects in EViews.
•
“Object View and Procedure Reference” provides a cross-referenced listing of the commands associated with each object, along with individual entries describing the syntax of each object command.
•
Appendix A. “Graph Creation Commands” documents specialized object commands for producing graph views from various EViews data objects.
•
Appendix B. “Object Command Summary” offers an alternative indexing of the object views and procedures discussed in the first two sections, pairing each object command with a list of the objects to which it may be applied.
