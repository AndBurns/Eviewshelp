@rweib
Element Functions
Weibull distribution random draw.
Syntax: @rtdist(m, a)
m: number,
a: number,
Return: number
Draw a random value from the Weibull distribution with probability density function,
for and 0 elsewhere.
Examples
= @rweib(1, 1)
returns a random draw from the Weibull(1, 1) distribution.
