Functional Coefficient Regression
The following discussion describes EViews tools for estimating and working with functional coefficients regression models. Functional coefficient regression is a semi-parametric approach to extend the standard regression framework to allow for non-linear, dynamic coefficients in regression estimation (Fan and Gijbels (1996); Cai, Fan, and Yao (2000)).
