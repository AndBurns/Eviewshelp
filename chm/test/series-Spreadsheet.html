Spreadsheet
The spreadsheet view is the basic tabular view for the series data and displays the raw, mapped, or transformed data series data in spreadsheet format.
You may customize your spreadsheet view in a number of ways (see “Changing the Spreadsheet Display” in ”Data Objects”).
The spreadsheet view may also be used to edit your series values (see “Editing a series” and “Series Adjust” for discussion).
In addition, the right-mouse button menu allows you to write the contents of the spreadsheet view to a CSV, tab-delimited ASCII text, RTF, HTML, Windows Metafile, PDF, LaTeX, or Markdown file. Simply right-mouse button click, select the Save table to disk... menu item, and fill out the resulting dialog.
